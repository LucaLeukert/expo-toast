## Queue options

- `maxVisible`: visible toast count per edge
- `maxQueue`: pending toast count per edge
- `dropPolicy`: `'oldest' | 'newest'` when pending queue is full

## Accessibility and motion

- `accessibilityLabel?: string`
- `announce?: boolean`
- `importance?: 'low' | 'normal' | 'high'`
- `motion?: 'system' | 'full' | 'minimal'`
